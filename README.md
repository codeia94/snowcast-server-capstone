# SNOWCAST

## Instruction and Installation
***

### Visual Studio Code
```
$ cd into the folder
$ npm i knex
$ npm i dotenv
```

Make sure to create a ```.env```` file

$ npx knex migrate:latest
$ npx knex seed:run

$ node --watch index.js
```

### MySql
```

Ensure that MySql is connected

```
mysql -u root -p
```

Enter password

Create a database
```
CREATE database mountains;

USE mountains;

show tables;
```
